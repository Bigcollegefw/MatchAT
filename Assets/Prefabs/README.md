# Tile Prefab Setup

To create the Tile prefab:

1. In Unity, right-click in Project window
2. Create > Prefab
3. Name it "Tile"
4. Add components to the prefab:
   - Image component (for the colored square)
   - Tile.cs script
   - TileInputHandler.cs script
5. Set the Image color to white (actual color comes from Tile.cs)
6. Set the Image size to 100x100 pixels
7. The Tile prefab should be placed at `Assets/Prefabs/Tile.prefab`
